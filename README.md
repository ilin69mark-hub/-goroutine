# Программа с горутинами на Go

Эта программа демонстрирует использование горутин и синхронизацию с помощью WaitGroup в Go.

## Описание задачи

Программа запускает 5 горутин, каждая из которых печатает свой порядковый номер 10 раз. Для синхронизации используется один вызов `wg.Add(5)`.

## Структура проекта

```
/workspace/
├── go.mod
├── main.go
└── README.md
```

## Как запустить

1. Убедитесь, что у вас установлен Go (версия 1.19 или выше)
2. Склонируйте или скопируйте этот проект
3. Выполните команду в терминале:

```bash
cd /workspace
go mod tidy  # если нужно обновить зависимости
go run main.go
```

Также можно собрать бинарный файл и затем запустить:

```bash
go build -o main main.go
./main
```

## Особенности реализации

- Используется один вызов `wg.Add(5)` перед запуском горутин
- Каждая горутина выводит свой порядковый номер и номер итерации
- Используется замыкание с передачей параметра в анонимную функцию, чтобы избежать захвата переменной цикла
- Все горутины синхронизированы с помощью `sync.WaitGroup`
